# daiki_beppu (別府 大貴)

**AI 時代のコード品質** を軸に、フロントエンドと開発者体験 (DX) を設計するエンジニア。
鳶職からの独学転職でエンジニア歴 1.5 年。BtoB 業務アプリの FE 基盤設計・実装、
DX 改善、AI 駆動開発ワークフローの整備などを担当しています。

## 🎯 軸

AI の出力を仕組みで担保・改善する実装技術として、次の 2 つを深めています。

- **AI × テスト** — テストを AI のガードレールとして設計。「テストが揃えば AI にタスクを安全に任せられる」状態を作る
- **AI × リファクタリング** — TDD を再実装する感覚で、AI に継続的なコード改善を任せられるワークフローを育てる
